# 社区资源

面向读者的**一手资料**索引，收录来自社区的与本案相关的资料——法庭文件、公开记录、官方声明及视频报道。目的是汇集人们所引用的**一手**资料（其他渠道的报道大多依赖*二手来源*），以便直接引用。本目录旨在**持续扩充**：新资源可添加至以下各节，其来源记录于*资源来源*中。**仅提供链接，不转载存档**（版权问题 + 编译时无网络出口）。最后审阅时间：**2026-06-03**。

> ⚠ **偏向性与来源说明。** 目前收集的资料主要来自 **r/RecklessBen**，这是一个**倾向辩护方**（支持 Schneider / 支持 Mansell）的社区，以下大多数托管地址为**社区镜像**（Google Drive、Proton Drive、Pastebin、Imgur），并非官方法院/机构系统。请将每条资料视为**线索**而非权威来源：法院文件应与书记员记录（犹他州 **XChange** / 俄勒冈州 **eCourt**，参见 [`../lawsuit/court-documents.md`](../lawsuit/court-documents.md)）进行核对，每项陈述在引用前应与原始平台核实。镜像内容可能被编辑或消失。随着其他视角的资源不断加入，也请注明其倾向性。
>
> ⚠ **个人信息保护 / 禁止人肉搜索。** 继续排除出现在这些文件、评论或视频帧中的任何个人信息/PII 内容——与本存档 [`../DISCLAIMER.md`](../DISCLAIMER.md) 保持一致。具体涉及未打码的随身摄像头录像，详见下方*依政策排除的内容*。
