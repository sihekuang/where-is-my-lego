# 社区来源

面向读者的**一手来源**索引，汇集自社区，涵盖本案相关的法庭文件、公开记录、官方声明及视频报道。其目的在于收集人们所引用的**一手**资料（其他地方的大量报道依赖*二手来源*），以便直接引用。本目录**持续扩充**：新来源可添加至以下各节，其来源出处记录于*这些来源的来历*中。**仅提供链接，不进行转载**（版权原因 + 编译时无网络出口）。最后审阅日期：**2026-06-03**。

> ⚠ **立场与来源说明。** 目前收集的资料大量来自 **r/RecklessBen**，这是一个**偏向辩护方**（支持 Schneider / 支持 Mansell）的社区，且以下大多数托管方为**社区镜像**（Google Drive、Proton Drive、Pastebin、Imgur），而非官方法庭/机构系统。请将每一项视为**线索**，而非权威依据：法庭文件须与书记员记录核实（犹他州 **XChange** / 俄勒冈州 **eCourt**，参见 [`../lawsuit/court-documents.md`](../lawsuit/court-documents.md)），每条陈述须在依赖前与原始平台核实。镜像内容可能被编辑或消失。随着来自其他视角的来源陆续添加，也请注明其立场倾向。
>
> ⚠ **个人信息（PII）/ 禁止人肉搜索。** 继续排除这些文件、评论或视频帧中出现的任何个人信息/PII内容——与本档案的 [`../DISCLAIMER.md`](../DISCLAIMER.md) 保持一致。具体关于未打码的随身摄像机录像，参见下文*依政策排除*部分。
