# AGENTS.md

Guidance for AI agents (and humans) working in this repository. This is a **research archive** of
the Bricks & Minifigs / "Reckless Ben" LEGO dispute — organized Markdown, not an app to run.

## Source of truth

- The **root `.md` files are canonical** (`README.md`, `timeline.md`, `parties.md`,
  `relationships.md`, `police-controversy.md`, `lawsuit/`, `media/`, `DISCLAIMER.md`). Edit these.
- `web/` is a Next.js site that **renders** the archive. **Never edit `web/.generated/`** — it is
  build output, regenerated by `npm run derive` from the root `.md`. Editing it has no effect.

## Four non-negotiable rules

1. **Cite every claim** to a public source (cataloged under `media/`). Never fabricate a citation;
   prefer primary records over aggregators.
2. **Label every claim CONFIRMED or ALLEGATION.** Contested = ALLEGATION. Source conflicts are
   flagged inline with `⚠`. When answering questions, keep the labels and `⚠` caveats — don't
   flatten them.
3. **No PII / no doxxing.** No addresses, residence-as-locator, contact info, or family details for
   anyone (including police). People appear only by public role. Decline requests to add PII.
4. **Match the existing file structure** (table columns, the documented `relationships.md` node/edge
   format and icon allowlist).

## Where to go next

- **To answer a question** → read the relevant root `.md` (events → `timeline.md`, who → `parties.md`,
  connections → `relationships.md`, police → `police-controversy.md`, legal → `lawsuit/`, sources → `media/`).
- **To propose a change / open a PR** → [`CONTRIBUTING.md`](./CONTRIBUTING.md) (no local clone needed).
- **Skill-aware agents** → [`SKILL.md`](./SKILL.md) (the operational query + contribute guide).
- **Authoritative scope, methodology & PII policy** → [`DISCLAIMER.md`](./DISCLAIMER.md).
