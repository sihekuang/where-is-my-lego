# 法庭文件 — 如何查找相关档案

与本档案其余部分一样，案件档案均以**链接形式呈现，而非直接托管**（编译环境无法访问外部网络，且法院/机构记录适用其自身的访问规则）。以下出现两类索引方式：

- **权威记录** — 文件正式存档所在的书记官/机构系统（犹他州 **XChange**、俄勒冈州 **eCourt / OJCIN**、**American Fork PD** 发布渠道）；以及
- **社区整理的镜像** — 社区收集的副本，已收录于[社区策划来源目录](../media/community-sources.md)。这些仅为**线索，不具权威性**：社区镜像可能被编辑或消失，且在编译期间**无法核实**其是否为经书记官认证的副本。在依赖任何镜像前，务必与官方记录进行交叉核实。

> 实际副本最为丰富的索引来源是
> **[社区策划来源目录](../media/community-sources.md)**（其§1涵盖法庭档案，§2涵盖 American Fork PD 记录）。该目录**偏向辩护方立场** — 详见其自身的偏向说明 — 且以便于引用而非中立性为组织原则。
